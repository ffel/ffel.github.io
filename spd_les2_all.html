Herhaling - Typen

int i = 1;
float f = 1.5;
String s = "string";
boolean b = true;

println(i, f, s, b);

Herhaling - Bewerkingen en Types

Een bewerking (delen, optellen) is anders, anders, anders voor ieder type

println( 2 + 4 );
println( "2" + "4" );

println( 7 / 2 );
println( 7.0 / 2.0 );

println( 7 / 2.0 );
println( 7.0 / 2 );

Herhaling - Toekenning en Gebruik van Variabelen

int a, b, c;

a = 1;
b = 2;

c = a + b;

println(a, b, c);

c = c + 4;

println(a, b, c);

a + c = b;  // fout!!

Herhaling - Oh Ja, modulo

println( 16 % 5 );

Herhaling - Laat de computer het werk doen

println( "72 / 9 = 8" );
println( "72 / 9 = " + 72 / 9 );

int a = 72; int b = 9;
println( a + " / " + b + " = " + 72 / 9 );

println( a, "/", b, "=", 72 / 9);
