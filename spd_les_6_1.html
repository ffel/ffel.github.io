Opgave 7.1 - Dambord

  • code netjes geformateerd
  • beperkt aantal globale variabelen
  • alle globale variabelen (zo mogelijk) final
  • functie tekenBord() met dubbele for loop.
  • setup() heeft size() en roept tekenBord()
  • slimme oplossing voor de kleuren
  • if (x%2 == y%2) ...
  • draw() voert geen werk uit

Opgave 7.1 - Uitwerking (1)

final int size = 40;
final int fields = 10;
final int DARK = #333333;
final int LIGHT = #DDDDDD;

void setup() {
  size(fields*size+1, fields*size+1);

  tekenBord();
}

Opgave 7.1 - Uitwerking (2)

void tekenBord(){
  for (int x=0; x<fields; x++) {
    for (int y=0; y<fields; y++) {
      fill(LIGHT);
      if (x%2 == y%2)
        fill(DARK);
        
      rect(x*size, y*size, size, size);
    }
  }
}

Opgave 7.2 - Uitwerking (2)

void plaatsStenen(int[][] bord){
  for (int x=0; x<bord.length; x++) {
    for (int y=0; y<bord[x].length; y++) {
      if (bord[x][y] == 1)
        plaatsSteen(x, y, 255);
      else if (bord[x][y] == 2)
        plaatsSteen(x, y, 0);
    }
  }
}

void plaatsSteen(int x, int y, int kleur) {
  fill(kleur);
  ellipse(x*size+2, y*size+2, size-4, size-4);
}

Opgave 7.3 (Geldwisselmachine)

  • functie int[] deelOp(int bedrag, int[] waarden)
  • in deelOp() eenvoudige toepassing van int rekenregels
  • in deelOp() maak je een nieuwe array int[] aantallen = new int[waarden.length]
  • aparte void tekenVerdeling(int[] waarden, int[] aantallen)

Opgave 7.3 - Uitwerking (1)

void setup() {
  int waarde = 3335;
  int[] waarden = {500, 100, 25, 1};
  println(deelOp(waarde, waarden));
}

int[] deelOp(int bedrag, int[] waarden) {

  int[] resultaat = new int[waarden.length];
  
  for (int i=0; i<waarden.length; i++) {
    int aantal = bedrag / waarden[i];
    resultaat[i] = aantal;
    bedrag -= aantal*waarden[i];
  }

  return resultaat;
}

Analyse

Je kan BP niet halen zonder een analyse!!!

  1. Schetsen

    • Begin situatie, handelingen, reacties, eindsituatie

    • Verzamel "uitdagingen" (om niet meteen een plan te hoeven maken)

  2. Stappenplan

    Voor iedere stap:

    • wat wil je bereiken
    • testcases (dit zijn eisen)
    • functies
    • events

  3. Bouw één stap

    • op basis van stappenplan
    • welk tabblad?

  4. Beschrijf tegenslagen

    • pas evt plan aan.
