Arrays (toch moeilijk ...)

int waarde = 3;          // waarde IS een een getal 
int[] arr = {1, 2, 3};   // arr BEVAT getallen

Je kan arr vergelijken met een kast.

Type int[] geeft aan dat er getallen in deze kast zitten.

int[] arr;

Is alsof je een plek in de kamer aanwijst en zegt:

hier komt mijn nieuwe kast.

arr = new int[10];

Dit is de meest algemene manier om de nieuwe kast te plaatsen. De kast bevat 10 standaard boeken.

arr verwijst nu naar een kast met 10 boeken

Arrays (7)

De drie stappen

int[] arr;                // declarerer
arr = new int[10];        // creëren
arr[0] = 5; arr[3] = 17;  // initieren

is compacter te schrijven als

int[] arr = new int[10];
arr[0] = 5; arr[3] = 17;

of zelfs

java int[] arr = {5,0,0,17,0,0,0,0,0,0};

Arrays (8)

Je kan natuurlijk ook een functie hebben die een array oplevert.

In plaats van

int[] arr;                // declarerer
arr = new int[10];        // creëren
arr[0] = 5; arr[3] = 17;  // initieren

krijg je dan iets als

double[] arr = giveMeSomeArray();

println(arr[2]); // 3.14

Arrays (9)

Je kan aan de array vragen hoe groot deze is

println( arr.length );  // 10

Dit kan je gebruiken om het laatste getal te lezen (of schrijven)

println( arr[ arr.length - 1 ] );  // 0

Of de voorlaatste

println( arr[ arr.length - 2 ] );  // 0

Arrays (10)

Waarom een array gebruiken?

  • Een reeks van waarden

    • Je kan gebruik maken van de volgorde

  • Gemakkelijk langs lopen (met een for)

    • je kan allerlei truucs uithalen met een reeks (loops)

      veel gemakkelijke dan losse variabelen

  • De lengte van de array hoeft pas bij creëren bekend te zijn.

    • dit kan niet eens met variabelen

Arrays (11)

Een twee dimensionale array is als een kast met meer planken.

float[][] arr2 = new float[3][10];

Een kast met 3 "planken" en 10 "posities" per plank 10 plekken per plank, drie planken, op iedere positie een float.

arr2;       // dit is de "kast" met floats  (een float[][] )
arr2[2];    //  dit is de onderste "plank" (een float[] )
arr2[2][9]; // dit is het "boek" rechts op de onderste plank

Arrays (12)

Om ieder element in een 2D array een waarde te geven heb je een dubbele loop nodig.

1
2
3
4
5
6
7

int[][] arr;                           // declaratie
arr = new int[4][5];                   // creatie
for (int i=0; i<arr.length; i++) {     // initialisatie
    for (int j=0; j<arr[i].length; j++) {
        arr[i][j] = i*j;
    }
}
  • regel 3: arr.length: aantal "planken" in "kast"
  • regel 4: arr[i].length: aantal "plekken" op "plank i"
  • regel 5: arr[i][j]: "boek" op "plank i" en "plek j"

ieder element in arr moet individueel worden gezet je kan niet meerdere waarden in arr in één keer een waarde geven.
