Opgave

Gegeven

int screen_w = 800;
int screen_h = 600;

int marge = 20;

int rechts_x, onder_y, midden_x, midden_y;

Vul de code aan:

  1. rechts_x = ....; zodat deze marge links van de rechterkant zit.

  2. onder_y = ....; zonder deze marge boven de onderkant zit.

  3. midden_x = .....; midden_y = ....; zodat deze in het midden ligt.
