  • Wanneer een waarde vaak wordt gebruikt (marge)
  • Wanneer je wilt voorbereiden op user input
    • of config file, of data base
  • Wanneer je code beter leesbaar wordt (tussenresultaat)

Toets - rectMode

CENTER mode:

  • de eerste twee waarden zijn het centrum van de rechthoek

Toets - te laat constrain

int windowW = 600;
int windowH = windowW * 2/3;
windowW = constrain(windowW, 400, 800*3/2);

Toets - Constrain

  • als niets wordt gezegd over minimale en maximale grootte: geen constrain
  • int breedte = constrain(600, 400, 800); is raar,
  • int w = 600; int breedte = constrain(w, 400, 600); is logischer
  • De mooiste: constrain(200, 100, pow(10, 999999))
    • zandkorrels: pow(10, 19);
    • sterren: pow(10, 23);
    • atomen: pow(10, 80);
