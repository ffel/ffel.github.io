Herhaling - Bewerkingen en Types

Een bewerking (delen, optellen) is anders, anders, anders voor ieder type

println( 2 + 4 );
println( "2" + "4" );

println( 7 / 2 );
println( 7.0 / 2.0 );

println( 7 / 2.0 );
println( 7.0 / 2 );

Herhaling - Toekenning en Gebruik

int a, b, c;

a = 1;
b = 2;

c = a + b;

println(a, b, c);

c = c + 4;

println(a, b, c);

a + c = b;  // fout!!
