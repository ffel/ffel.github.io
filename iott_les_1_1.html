Hello World

#include <stdio.h>

/* print een bericht voor de klas */
main() {
    printf("Hallo %s!\n", "IoT klas");
}

Compileren

$ gcc -Wall hallo.c -o hallo

Opgave

  • Compileer hallo.c en draai deze.

  • Experimenteer met de code en het gcc command en let op welke fouten je krijgt.

Code in meerdere bestanden

main.c:

main() {
    greet();
}

greet.c:

void greet()
{
    printf("hallo klas!\n");
}

Opgave

compileer met:

$ gcc main.c greet.c

  • Hoe kan je de code in de bestanden aanpassen zodat deze compileert?

    • -Wall helpt wellicht een beetje

Antwoord - Header Files

greet.h:

#include <stdio.h>

void greet();

greet.c:

#include "greet.h"

void greet()
{
    printf("hallo klas!\n");
}

Antwoord (2)

main.c:

#include "greet.h"

main() {
    greet();
}

Header Files

  • Een h file bevat dat gedeelte van de bijbehorende c file dat je beschikbaar wilt maken.

  • Er is een complicatie.

Lees een h file maximaal één keer in

greet.h:

#ifdef greet_h
#define greet_h

#include <stdio.h>

void greet();

#endif
  • Ook wordt #pragma once gebruikt.

Opgave

  • Compileer nu het vorige voorbeeld.

Compileren

$ gcc -c greet.c
$ gcc -c main.c
$ gcc main.o greet.o -o hoi

Integer waarden

#include <stdio.h>

int main()
{
    char c;
    short s;
    int i;
    long l;
    unsigned long ul;
    long long ll;

    printf("c: %zu, s: %zu, i: %zu, l: %zu, ul: %zu, ll: %zu",
       sizeof(c), sizeof(s), sizeof(i), sizeof(l),
       sizeof(ul), sizeof(ll));
}

Integer waarden (2)

  • aantal bytes verschilt van architectuur en compiler (?)

  • waarom niet gewoon altijd long long?

Daarom

#include <stdlib.h>
#include <stdio.h>

int main()
{
    int8_t i8;
    int16_t i16;
    int32_t i32;
    int64_t i64;

    printf("i8: %zu, i16: %zu, i32: %zu, i64: %zu\n",
       sizeof(i8), sizeof(i16), sizeof(i32), sizeof(i64));
}

Aside - Preprocessor

  • We hebben al een aantal #include, #define etc gezien.

  • Dit zijn instructies voor de pre processor

Define vs Const

#include <stdio.h>

#define QUESTION "What is the meaning of life?"
const int answer = 42;

int main()
{
    printf("Q: %s, A: %d\n", QUESTION, answer);
}

  • Strings zijn lastiger, en #define helpt soms een beetje

  • const respecteert scope.

Make

Een programma van één bestand kan je compileren met make:

Ons voorbeeld hallo.c:

$ make hallo
  • kijk naar de uitvoer!

Make opties

We vergeten steeds -Wall, toch?

Zet in Makefile

CFLAGS=-Wall

Make afhankelijkheden

main.c hing af van greet.c:

Dit kan je aangeven in je Makefile:
