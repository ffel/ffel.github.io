Herhaling - Functies

  • Een functie doet een klus voor je

    • teken een lijn blok
    • print een waarde
    • doet een instelling (fill(), smooth())

  • Een functie berekent een waarde voor je

    • max(1, 12, 3)

Herhaling - puntkomma

println( constrain(20, 10, 100) );

// fout:                       *
println( constrain(20, 10, 100); );

De ; sluit een statement af, een (geneste) instructie.

Herhaling - Declaratie, Initialisatie

Declaratie:

  • reserveer een stukje geheugen
  • zeg wat het programma er in kan stoppen
int a;

Initialisatie:

  • vul het stukje geheugen met een waarde
a = 3;
