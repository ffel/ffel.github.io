  1. Speelveld
    • size()
  2. Robot
    • rectangle, x, y, breedte, hoogte, kleur
  3. Vracht
    • ellipse, x, y, breedte (is breedte robot), hoogte
  4. Robot bewegen
    • pijltjestoetsen
    • niet buiten speelveld
    • verplaatsing gelijk aan stapgrootte robot
  5. Vracht vastpakken
    • enter toets
    • als robot op vracht staat
    • (alleen als robot vracht nog niet vast heeft)
  6. Vracht meenemen
    • met bewegen robot
    • alleen als robot vracht heeft vastgepakt
    • positie vracht is positie robot, verschil ellipse en rect!!
  7. Vracht loslaten
    • enter toets
    • alleen als robot vracht vast heeft

Toetsopgave (2)

int balkHoogte = 0;
int wBreedte = 400;
int marge = 50;

void setup() {
  background(0);
  size(wBreedte,wBreedte); 
}

void draw() {
  fill(#E109E5);
  rect(marge, wBreedte - balkHoogte, wBreedte - marge*2, balkHoogte); // beginpositie wordt aangepast
  balkHoogte++;
}
