Bespreken tot en met 1.7

Japanse Vlag - Vars

Dankzij Analyse

int schermW = 500;
int schermH = schermW / 3 * 2;
int marge = 25;

int vlagW = schermW - 2*marge;
int vlagH = schermH - 2*marge;

int vlagX = marge;
int vlagY = marge;

int roodX = vlagX + vlagW / 2;
int roodY = vlagY + vlagH / 2;

int roodDiam = vlagW / 5;

String land = "Nippon";

Japanse Vlag - Overige

size(schermW, schermH);
smooth();
background(0,0,0);

fill(255,255,255);
rect(vlagX, vlagY, vlagW, vlagH);

fill(255,0,0);
ellipse(roodX, roodY, roodDiam, roodDiam);

fill(0,0,0);
textAlign(CENTER, CENTER);
text(land, roodX, roodY);

Herhaling - Bewerkingen en Types

Een bewerking (delen, optellen) is anders, anders, anders voor ieder type

println( 2 + 4 );
println( "2" + "4" );

println( 7 / 2 );
println( 7.0 / 2.0 );

println( 7 / 2.0 );
println( 7.0 / 2 );

Herhaling - Toekenning en Gebruik van Variabelen

int a, b, c;

a = 1;
b = 2;

c = a + b;

println(a, b, c);

c = c + 4;

println(a, b, c);

a + c = b;  // fout!!

Herhaling - Oh Ja, modulo

println( 16 % 5 );
println( 24 % 7 );
println( 30 % 5 );

Herhaling - Laat de computer het werk doen

println( "72 / 9 = 8" );
println( "72 / 9 = " + 72 / 9 );

int a = 72; int b = 9;
println( a + " / " + b + " = " + a / b );

println( a, "/", b, "=", a / b );
