Aanwezigheidsregistratie

Programma

  1. Je krijgt code op basis van afgelopen woensdag

  2. Bestudeer deze code

  3. Zelf testcode maken voor onderdelen

  4. Zelf statemachines implementeren voor contineous en interval

  5. Opdracht afronden

Volgorde implementatie

  1. Zorg dat je helder hebt wat je moet bouwen (functioneel).

    (Schetsen, Lijst met Requirements, MoSCoW)

  2. Maak eerste ontwerp (Software Architectuur, State Diagrammen)

  3. Sluit componenten aan en bouw de onderste laag (controllers) van je SArch.

    Naast je onderste laag heb je nu alleen nog maar main waarin je de onderste laag test.

  4. Bouw van onder af de andere lagen, update je ontwerp.

    Je kan je opstelling testen aan de hand van je Lijst met Requirements.

Verder met weekopdracht

  • We hebben een ontwerp.

  • Code die je krijgt bevat de onderste laag van de SArch.

    Er zitten wat wijzigingen in de code.

  • Voor je gaat met bouwen, eerst een aantal vragen over de code.
