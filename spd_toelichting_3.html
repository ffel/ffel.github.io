Toelichting

Een aantal mensen loopt vast in de opgaven van module 3.

Hier een aantal tips waarmee ik hoop dat je verder kunt.

Basisstructuur

De basisstructuur voor een Sketch applicatie is:

// definieer hier je "globale variabelen", dat zijn
// variabelen die overal in je programma beschikbaar
// zijn
int screenW = 300;
int screenH = screenW;

// setup() wordt éénmaal uitgevoerd,
// hierin kan je bijvoorbeeld size zetten
void setup() {
  size(screenW, screenH);
}

// draw() wordt meerdere keren per seconden uitgevoerd
// hiermee voeg je nieuwe onderdelen aan het scherm toe
void draw() {
  // zet de schermkleur
  background(255);
}

Muis

Naast setup() en draw() heb je ook functies waarmee je de muis kan "lezen".

// mouseClicked() wordt telkens uitgevoerd wanneer je
// de muis klikt
void mouseClicked() {
  // de positie van de muis is te lezen uit mouseX en
  // mouseY
  println("muis klik", mouseX, mouseY);
}

// mouseMoved() wordt telkens uitgevoerd wanneer je de
// muis beweegt, voor ieder stukje bewegen een keer (vaak dus)
void mouseMoved() {
  // de positie van de muis is te lezen uit mouseX en
  // mouseY
  println("mouse", mouseX, mouseY);
}

Toetsenbord

Er zijn ook functies waarmee je toetsaanslagen kan "lezen".

// keyPressed() wordt uitgevoerd wanneer je een toets op
// het toetenbord wordt ingedrukt.
void keyPressed() {
  // "normale" letters staan in "key"
  println(key);
  // voor "bijzondere" toetsen heb je "keyCode" nodig
  switch(keyCode) {
    case LEFT:
      println("links");
      break;
    case UP:
      println("omhoog");
      break;
    case ENTER:
      println("enter");
      break;
  }
}

Voorgedefinieerde variabelen

Naast je eigen variabelen zijn er ook standaard variabelen die Processing een waarde geeft.

Jij kan deze waarde lezen in je programma.

  • width en height geven de grootte van je scherm
  • mouseX en mouseY geven de positie van je muis
  • key en keyCode gebruik je om te bepalen welke toetsen zijn ingedrukt.

Condities

In deze opgaven gaat het vooral om gebruik van if-else.

Alle condities kan je met if-else controleren

int val = 12;

if (val > 10) {
    // voer uit wanneer val groter is dan 10
}
else {
    // voer uit in alle andere gevallen
}

En Of

Condities kunnen samengesteld worden

int val = 12;

if (val < 10 || val > 100) {
    // voer uit wanneer val kleiner is dan 10 OF
    // val groter is dan 100
}
else if (val > 50 && val < 60) {
    // voer uit wanneer val groter is dan 50 EN
    // val kleiner is dan 60
}

Switch

Wanneer in een if steeds dezelfde waarde wordt gecontroleerd, bijvoorbeeld

int val = 1;

if (val == 0) {
    // voer uit wanneer val gelijk is aan 0
}
else if (val == 1) {
    // voer uit wanneer val gelijk is aan 1
}
else if (val == 2) {
    // voer uit wanneer val gelijk is aan 2
}
else {
    // voer uit in alle andere gevallen
}

dan kan je ook de switch gebruiken die iets compacter is.

Switch (2)

De vorige code maar nu met switch:

int val = 1;

switch(val) {
    case 0:
        // voer uit wanneer val gelijk is aan 0
        break;
    case 1:
        // voer uit wanneer val gelijk is aan 1
        break;
    case 2:
        // voer uit wanneer val gelijk is aan 2
        break;
    default:
        // voer uit in alle andere gevallen
}

Hints bij Opgaven (2)

Opgaven 3.2 en verder

Met alles in deze toelichting moet je een eind kunnen komen. Je moet alleen nog weten hoe je berekent of een getal even of oneven is.

int val = 13;

if (val % 2 == 0) {
    println(val + " is even")
}

if (val % 2 == 1) {
    println(val + " is oneven")
}
